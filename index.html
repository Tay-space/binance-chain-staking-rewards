What is this?

You can use this website to export your Binance Chain Staking rewards to a file in the CSV file format.

The CSV file uses Koinly's custom CSV format but can be easily adapted to other formats to make it compatible with alternative cryptocurrency tax services.

Usage

  • Enter your Binance Chain Network address (starting with "bnb") into the input field above.
  • Click the "Export transactions to CSV" button.

You will be presented with a CSV download containing the full Binance Chain Staking reward history for the given address.

Is this safe to use?

To put it simply, yes.

  • It's not possible to perform a meaningful attack with only a public address.
  • The code for this website is open source and can be viewed in full via the GitHub repository.
  • The export process takes place in client, i.e. no data is sent back to the server. In fact, this website is hosted on GitHub pages which doesn't provide access to the backend server.
